# Setara Kids School Website

This repository contains the static HTML, CSS, and JavaScript files for the Setara Kids School website.

Structure:
- index.html              : Homepage with carousel and program highlights
- classes.html            : Page describing learning programs and featured activities
- program-kb-kecil.html   : Details for "Kelompok Bermain Kecil" (ages 2-3) with activity cards
- program-kb-besar.html   : Details for "Kelompok Bermain Besar" (ages 3-4) with activity cards
- Other HTML pages include facility.html, team.html, testimonial.html, contact.html, login.html, etc.
- api/                    : PHP backend scripts for database and authentication (used by the site)
- css/, js/, img/, lib/   : Supporting assets.

How to view:
Open the HTML files in a browser or host them under a PHP/HTTP server such as Laragon (already configured). The site uses Bootstrap 5 and some custom CSS.

To Edit:
- HTML pages: modify directly or use templating as desired.
- CSS: main custom rules are located in css/style.css and inline <style> sections within pages.
